# [LeetCode][leetcode] task # 172: [Factorial Trailing Zeroes][task]

Description
-----------

> Given an integer `n`, return _the number of trailing zeroes in `n!`_.
> 
> Note that `n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1`.

Example
-------

```sh
Input: n = 5
Output: 1
Explanation: 5! = 120, one trailing zero.
```
