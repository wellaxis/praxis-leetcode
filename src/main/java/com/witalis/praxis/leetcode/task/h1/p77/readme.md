# [LeetCode][leetcode] task # 77: [Combinations][task]

Description
-----------

> Given two integers `n` and `k`, return
> _all possible combinations of `k` numbers out of the range `[1, n]`_.
> 
> You may return the answer in **any order**.

Example
-------

```sh
Input: n = 4, k = 2
Output:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```
