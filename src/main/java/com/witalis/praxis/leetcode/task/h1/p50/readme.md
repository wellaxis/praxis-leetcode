# [LeetCode][leetcode] task # 50: [Pow(x, n)][task]

Description
-----------

> Implement pow(x, n), which calculates x raised to the power n (i.e., x^n).

Example
-------

```sh
Input: x = 2.10000, n = 3
Output: 9.26100

Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
```
