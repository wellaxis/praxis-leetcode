# [LeetCode][leetcode] task # 338: [Counting Bits][task]

Description
-----------

> Given an integer `n`, return _an array `ans` of length `n + 1` such that for each `i` (`0 <= i <= n`),
> `ans[i]` is the **number of** `1`**'s** in the binary representation of `i`.

Example
-------

```sh
Input: n = 5
Output: [0,1,1,2,1,2]
Explanation:
0 --> 0
1 --> 1
2 --> 10
3 --> 11
4 --> 100
5 --> 101
```
