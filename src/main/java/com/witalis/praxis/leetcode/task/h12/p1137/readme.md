# [LeetCode][leetcode] task # 1137: [N-th Tribonacci Number][task]

Description
-----------

> The Tribonacci sequence T[n] is defined as follows:
> 
> T[0] = 0, T[1] = 1, T[2] = 1, and T[n+3] = T[n] + T[n+1] + T[n+2] for n >= 0.
> 
> Given `n`, return _the value of T[n]_.

Example
-------

```sh
Input: n = 4
Output: 4
Explanation:
    T_3 = 0 + 1 + 1 = 2
    T_4 = 1 + 1 + 2 = 4
```
